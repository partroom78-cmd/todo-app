# Tasky · 할 일 관리

순수 HTML/CSS/JavaScript로 만든 개인용 할 일 관리 웹페이지입니다.
별도 설치나 서버 없이 브라우저에서 바로 실행됩니다.

## 주요 기능

- 할 일 추가 / 수정 / 삭제 / 완료 체크
- 카테고리 분류: 업무 / 개인 / 공부
- 마감일 입력 및 색상 표시 (기한 초과 · 오늘 · 내일 · 여유)
- 상단 요약 카드: 전체 / 진행 중 / 완료 / 완료율
- 상태 필터(전체 · 진행 중 · 완료) + 카테고리 필터
- 실시간 검색, 정렬(등록순 · 마감일 임박순 · 카테고리순)
- 완료 항목 일괄 삭제
- 모바일 반응형 UI
- 브라우저 `localStorage`에 자동 저장 (새로고침·재방문 후 유지)

## 실행 방법

`index.html` 파일을 웹 브라우저에서 열면 됩니다. 추가 설치가 필요 없습니다.

## 기술 스택

- HTML, CSS, 순수 JavaScript (Vanilla JS)
- 외부 라이브러리·프레임워크·빌드 도구 없음
- 데이터 저장: 브라우저 localStorage

## 파일 구성

| 파일 | 설명 |
|------|------|
| `index.html` | 화면 구조 |
| `style.css` | 스타일 (대시보드 톤, 반응형) |
| `script.js` | 기능 로직 (CRUD · 필터 · 정렬 · 통계 · 저장) |
| `PRD.md` | 제품 요구사항 |
| `PLAN.md` | 구현 계획 |
| `CLAUDE.md` | 개발 참고 문서 |
